# CS6360-database

Database project for CS6360

Program is only compatible with python 3
Required libraries are listed in the requirements folder.

commands supported:
	CREATE
	DEBUG
	DELETE
	DROP
	EOF
	EXIT
	HELP
	INSERT
	SELECT
	SHOW
	UPDATE

Value types supported:
	python primitives (namely INT and string as TEXT)
	Date
	DateTime
	Time
	Year
